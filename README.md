# Skill2Future – Career Recommendation

A simple prototype web application that recommends careers based on skills and interest. Built with React (Vite) and Node.js (Express).

## Tech Stack
- **Frontend:** React (Vite), plain CSS and Inter font
- **Backend:** Node.js, Express, CORS

## Features
- Enter skills and select an interest from a dropdown.
- Receive tailored career recommendations with match percentages.
- Beautiful, responsive UI.

## How to Run Locally

### 1. Start the Backend Server
```bash
cd backend
npm install  # (if not already installed)
node server.js
```
The backend will run at `http://localhost:3000`.

### 2. Start the Frontend Application
Open a new terminal window:
```bash
cd frontend
npm install  # (if not already installed)
npm run dev
```
The frontend will start a Vite development server. Open the local URL (typically `http://localhost:5173/`) in your browser to view the application.
